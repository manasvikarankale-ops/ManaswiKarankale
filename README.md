This project is a Streamlit-based web application for stock price prediction using machine learning models.

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## Deployment
The application is deployed on *Streamlit Cloud* and is automatically updated on every push to the GitHub repository.
